Project Overview

JCOE 공정은 두꺼운 강판을 J, C, O 단계로 굽혀 원통관을 만드는 공정입니다. 성형 중 반지름과 최종 용접 개구 간격이 목표값에서 벗어나면 용접 불량과 구조적 약점으로 이어질 수 있습니다.

목표

성형 진행 중 형상을 계측하고 시각화하여 작업자가 공정 중 보정 판단을 할 수 있도록 구성했습니다.

수행 범위

C# 운영 시스템, Python Vision 연동, 측정 로직, UI 구성, 제어기 연동 협의, 현장 설치/검증, 운영자 교육을 수행했습니다.

Problem

공정 판단

  • 성형 상태를 작업자의 시각적 판단과 경험에 의존하는 구간이 존재했습니다.
  • 성형 완료 후 측정만으로는 공정 중 즉각적인 보정이 어렵습니다.
  • 제품별 형상 이력이 체계적으로 남지 않으면 이후 원인 분석과 공정 개선이 제한됩니다.

현장 계측

  • 카메라와 파이프 간 거리가 항상 동일하지 않아 픽셀 기반 측정값 보정이 필요했습니다.
  • 조명, 분진, 표면 상태, 배경에 따라 일반 영상처리만으로는 경계 검출이 불안정해질 수 있었습니다.
  • Windows 기반 운영 환경과 기존 설비 인터페이스를 함께 고려해야 했습니다.

핵심은 작업자의 감각을 대체하는 것이 아니라, 작업자가 다음 프레스 동작을 판단할 수 있는 수치와 추세를 보여주는 것이었습니다.

Measurement Structure

성형 중에는 카메라 기반 반지름 측정으로 STEP별 형상 변화를 확인하고, 성형 완료 후에는 2D 레이저 프로파일러로 Open Gap을 길이 방향으로 측정합니다.

카메라 측정 영역 프레스 상·하부 모듈, 블레이드, 강판 입력 방향, 좌우 카메라 측정 영역
2D 레이저 Gap 측정 레이저 프로파일러, 엔코더 옵션, 위치 기준 Gap Trend 저장

Camera Measurement

카메라 기반 반지름 측정은 성형 과정에서 파이프 외곽을 촬영하고, AI 기반 경계 검출 및 보정 로직을 통해 반지름 편차를 계산하는 구조입니다.

영상 입력

각 STEP에서 카메라 영상을 받아 성형 상태의 외곽 정보를 추출합니다.

AI Edge Detection

조명과 배경 영향을 줄이기 위해 Python 기반 이미지 분할/경계 검출을 연동했습니다.

Radius Offset

목표 반경 대비 편차를 계산하고 허용 범위 이탈 시 UI에서 즉시 확인할 수 있게 했습니다.

Control Link

측정 시스템은 프레스 제어 흐름과 연동되어 STEP별 촬영과 기록을 수행합니다. 좌·우 성형 STEP을 구분하고 설비 이벤트에 맞춰 측정 흐름을 이어가도록 구성했습니다.

좌측 성형 STEP 연동 프레스 동작 신호와 촬영/기록 흐름 동기화
우측 성형 STEP 연동 반대 방향 성형 단계의 STEP 이벤트 처리
Gap 측정 동작 성형 완료 후 Open Gap 측정과 그래프 전환

Gap Measurement

Open Gap 측정은 성형이 끝난 뒤 용접 준비 간격을 확인하는 단계입니다. 2D 레이저 프로파일러가 파이프 길이 방향으로 단면 프로필을 연속 측정하고, 위치 기준 Gap 데이터로 저장합니다.

측정 방식

  • 파이프 상부 개구부를 비접촉 방식으로 연속 스캔합니다.
  • 두 판재 모서리 사이의 간격을 계산해 용접 준비 상태를 확인합니다.
  • 엔코더를 사용할 경우 길이 좌표와 Gap 값을 함께 기록합니다.

운영 활용

  • Gap 그래프를 저장하고 과거 기록과 비교할 수 있습니다.
  • 용접부 전체 길이에 걸친 Gap 균일도를 확인할 수 있습니다.
  • 이상 구간을 품질 이력과 연결할 수 있습니다.

System Flow

1. STEP 이벤트프레스 제어 흐름에서 촬영/기록 시점 인식
2. 영상 수집카메라가 성형 중 외곽 영상을 취득
3. 경계 검출Python Vision이 파이프 영역과 외곽 후보 추출
4. 보정/계산거리 보정과 반지름 편차 계산
5. Gap 스캔성형 완료 후 Open Gap을 길이 방향으로 측정
6. UI/이력화그래프, 알람, 작업 조건, 측정 이력 저장
